Particle Controls:
Left Click: To add a particle emitter at the current mouse position
E: To toggle drawing of particle bounds
Q: to toggle particle/rigid-shape collisions
Left/Right Key: switch between the different particles emitted

Shape Behavior Controls:
  • V: Motion on/off
  • F: Inject random velocity to all objects
  • C: Create new Rectangle/Circle (randomly)
Hero Controls:
  • WASD-ZX: Move/Rotate(ZX) Hero Object
  • Up/Down + MNF: Increase/Decrease Mass/Restitution/Friction
